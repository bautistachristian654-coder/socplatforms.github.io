## 🔗 Social Media Links

The website contains links to the following platforms:

* Facebook
* YouTube
* Instagram
* TikTok
* Email

Each link opens in a **new browser tab** using `target="_blank"`.

## 📷 Profile Image

The profile picture is displayed using the `<img>` tag.

Example:

```html
<img src="lloyd.jpg" alt="profile image" class="profile-img">
```

## 🚀 How to Run the Project

1. Download or clone the project files.
2. Make sure all files are in the same folder.
3. Open **index.html** in any web browser.
4. The website will load and display the profile with social links.
